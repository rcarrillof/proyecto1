Breadcrumb

Indique la ubicación de la página actual dentro de una jerarquía de navegación que agrega automáticamente separadores a través de CSS.

Ejemplo:

  1. Inicio
  1. Inicio
  2. Lista de componentes
  1. Inicio
  2. Lista de componentes
  3. Data

Dividers

Los divisores se agregan automáticamente en CSS a través de ::before y content.

  1. Inicio
  2. Library

También puede eliminar la configuración del divisor --bs-breadcrumb-divider: ''; (las cadenas vacías en las propiedades personalizadas de CSS cuentan como un valor), o establecer la variable Sass en $breadcrumb-divider: none;.
